PlaNet

Learning Latent Dynamics for Planning from Pixels — Hafner et al., ICML 2019

🔗 danijar.com/planet
💡
Core Ideas
🔄

Recurrent State Space Model (RSSM)

Combine deterministic recurrent path h_t = f(h_{t-1}, s_{t-1}, a_{t-1}) with stochastic latent s_t ~ p(s_t|h_t). Both paths essential — deterministic for memory, stochastic for uncertainty.

🎯

Online Planning with CEM

No actor/critic network. At each step, use Cross-Entropy Method to search for best action sequence: sample J=1000 candidates, evaluate by model rollout, refit to top K=100, repeat I=10 iterations. Execute only first action.

🔭

Latent Overshooting

Standard variational bound only trains 1-step transitions. Latent overshooting trains multi-step predictions (d=1..D steps ahead) in latent space, improving long-term accuracy.

🧩
Model Components (RSSM)

Deterministic Path (GRU)

\( h_t = f(h_{t-1}, s_{t-1}, a_{t-1}) \)

Provides memory for long-term dependencies via a GRU recurrent unit

Stochastic Prior

\( s_t \sim p(s_t \mid h_t) \)

Diagonal Gaussian. Captures environment uncertainty in the latent state

Posterior (Encoder)

\( q(s_t \mid h_{t-1}, a_{t-1}, o_t) \)

CNN encoder + MLP. Conditions on actual observation for better latent estimate

Observation Model

\( p(o_t \mid h_t, s_t) \)

Deconvolutional decoder. Reconstructs 64×64 images from the latent state

Reward Model

\( p(r_t \mid h_t, s_t) \)

MLP. Predicts scalar reward from latent state for planning evaluation

Training Objectives

Variational Bound (ELBO)

Standard 1-step variational bound: reconstruct observations + regularize posterior to match prior via KL divergence

Latent Overshooting

$$\mathcal{J}_O = \sum_{t=1}^T \sum_{d=1}^D \beta_d \left[\mathbb{E}[\ln p(o_t|s_{t|t-d})] - \text{KL}[q(s_t|o_{\leq t}) \| p(s_t|s_{t-d})]\right]$$

Multi-step prediction loss: trains d-step transitions p(s_t|s_{t-d}) against the same posterior, with weighting factors β_d for consistency

CEM Planning Objective

$$a_{1:H}^* = \arg\max_{a_{1:H}} \mathbb{E}_p\left[\sum_{\tau=t}^{t+H} r_\tau\right]$$

Find the action sequence that maximizes expected sum of predicted rewards over planning horizon H, solved approximately via CEM

Paper Lineage

Builds On

World Models
Ha & Schmidhuber, 2018 — Two-stage world model (VAE + MDN-RNN). PlaNet unifies into a single RSSM trained end-to-end.
E2C (Embed to Control)
Watter et al., 2015 — Locally linear latent dynamics model for control from images. PlaNet scales to richer tasks with recurrence.
Deep Kalman Filters
Krishnan et al., 2015 — Deep state space models with variational inference. PlaNet adds the deterministic path for reliable memory.
CEM
Rubinstein, 1997 — The Cross-Entropy Method for optimization. PlaNet uses it for online planning over learned dynamics.

Built Upon By

Dreamer
Hafner et al., 2020 — Replaces CEM planning with learned actor-critic in imagination. Same RSSM, but amortized policy is faster at test time.
DreamerV2
Hafner et al., 2021 — Discrete latent states (32×32 categoricals), KL balancing. Achieves human-level Atari from pixels.
DreamerV3
Hafner et al., 2023 — Fixed hyperparameters across domains, symlog predictions. Scales from Atari to Minecraft without tuning.
SOLAR
Zhang et al., 2019 — Stochastic latent actor-critic combining PlaNet-style world model with soft actor-critic for real-robot tasks.
