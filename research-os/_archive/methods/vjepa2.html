V-JEPA 2

Self-Supervised Video Models Enable Understanding, Prediction and Planning — Assran et al., 2025

🔗 github.com/facebookresearch/vjepa2
💡
Core Ideas
📈

Scaling Self-Supervised Video Pretraining

Scale along ALL axes simultaneously: data (2M→22M videos), model (300M→1B params), training (90K→252K iters), and progressive resolution (256px→384px, 16→64 frames). Each contributes +1–1.5 points.

🤖

Action-Conditioned World Model (V-JEPA 2-AC)

Freeze the video encoder, train a 300M-param autoregressive transformer predictor on top using only 62 hours of robot data. Block-causal attention predicts next-frame representations conditioned on actions.

🧭

Planning via Energy Minimization

Plan robot actions by minimizing L1 distance between predicted future state and goal state in representation space. No reward function needed — just provide a goal image.

🧩
Model Components

ViT-g Backbone (1B params)

Vision Transformer with 1B parameters trained on 22M videos. Uses 3D-RoPE instead of absolute position embeddings. Progressive training: 256px→384px, 16→64 frames.

Autoregressive Predictor (V-JEPA 2-AC)

300M params, 24 transformer layers, 16 heads, 1024 dim. Block-causal attention. Takes (features, actions, poses) → predicts next features.

Teacher-Forcing Loss

\( \mathcal{L}_{\text{TF}} = \frac{1}{T}\sum_{k=1}^{T}\|\hat{z}_k - z_k\|_1 \)

L1 between predicted and actual backbone features at each timestep

Rollout Loss

\( \mathcal{L}_{\text{rollout}} = \|P(a_{1:T}; z_0) - z_{T+1}\|_1 \)

L1 error over multi-step autoregressive rollouts. Helps reduce compounding errors.

Energy-Based Planning

\( E(a_{1:H}) = \|P_{\text{AC}}(z_0, s_0, a_{1:H}) - z_{\text{goal}}\|_1 \)

Minimize over action sequences. No reward function needed.

Training Objectives

Pretraining (V-JEPA Objective)

$$\mathcal{L}_{\text{pretrain}} = \|P_\phi(\Delta_y, E_\theta(x)) - \text{sg}(\bar{E}_{\bar\theta}(y))\|_1$$

Predict masked video features from visible context. The target encoder is an EMA of the online encoder with stop-gradient.

Teacher-Forcing Loss

$$\mathcal{L}_{\text{TF}}(\phi) = \frac{1}{T}\sum_{k=1}^{T}\left\|P_\phi\big((a_t, s_t, E(x_l))_{l\leq k}\big) - E(x_{k+1})\right\|_1$$

At each timestep, predict the next frame's features given all preceding frames, actions, and end-effector states.

Rollout Loss

$$\mathcal{L}_{\text{rollout}}(\phi) = \|P_\phi(a_{1:T}; s_1, z_1) - z_{T+1}\|_1$$

Multi-step autoregressive prediction loss. Reduces compounding errors during long-horizon rollouts.

Planning Energy

$$a_{1:H}^* = \arg\min_{a_{1:H}} \|P_{\text{AC}}(z_0, s_0, a_{1:H}) - z_{\text{goal}}\|_1$$

Find the action sequence that minimizes the distance between predicted future and goal in representation space. Only the first action is executed, then replanning occurs.

Paper Lineage

Builds On

V-JEPA
Bardes et al., 2024 — Original self-supervised video prediction in latent space. V-JEPA 2 scales the data, model, and training far beyond the original.
Scaling ViT
Zhai et al., 2022 — Foundational work on scaling Vision Transformers. V-JEPA 2 uses ViT-g (1B params) with 3D-RoPE.
Droid Dataset
Khazatsky et al., 2024 — 62 hours of robot manipulation data (7-DOF Franka Panda) used to train V-JEPA 2-AC.
I-JEPA
Assran et al., 2023 — Image-level JEPA that inspired the video extension. Predicts image features from context.
RoPE
Su et al., 2024 — Rotary Position Embedding extended to 3D (spatial + temporal) for video transformers.

Built Upon By

Frontier Paper
V-JEPA 2 is a very recent contribution (2025). Follow-up work building on this foundation is expected in robotics, video understanding, and world models.
