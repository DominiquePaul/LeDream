Soft Actor-Critic (SAC)

Maximum Entropy Reinforcement Learning Algorithm — Haarnoja et al., 2018

🔗 github.com/haarnoja/sac
🏗️
Network Architecture
📋
Algorithm Steps

Learning loop (per update). Acting + collecting transitions happens outside this loop.

1

Sample from Replay Buffer

Draw mini-batch of transitions (s, a, r, s', d)

2

Compute Target Q-value

Sample ã' ~ π(·|s'), evaluate targets, subtract entropy

3

Update Critics

Train independently via MSE to soft Bellman target

4

Update Actor

Sample ã ~ π(·|s), maximize Q − α·log π

5

Update Temperature α

Adjust entropy coefficient automatically

6

Soft Update Targets

φ̄ ← τφ + (1-τ)φ̄

📐
Loss Functions

Critic Loss (Soft Bellman)

\( J_Q(\phi) = \mathbb{E}_{(s,a,r,s') \sim \mathcal{D}} \left[ \frac{1}{2} \left( Q_\phi(s,a) - y \right)^2 \right] \)
\( y = r + \gamma (1-d) \left( \min_{j=1,2} Q_{\bar{\phi}_j}(s', \tilde{a}') - \alpha \log \pi_\theta(\tilde{a}'|s') \right) \)

Critics are trained independently (separate parameters + optimizer states)

Actor Loss

\( J_\pi(\theta) = \mathbb{E}_{s \sim \mathcal{D}, \tilde{a} \sim \pi_\theta} \left[ \alpha \log \pi_\theta(\tilde{a}|s) - \min_{j=1,2} Q_{\phi_j}(s, \tilde{a}) \right] \)

Temperature Loss

\( J(\alpha) = \mathbb{E}_{\tilde{a} \sim \pi_\theta} \left[ -\alpha \left( \log \pi_\theta(\tilde{a}|s) + \bar{\mathcal{H}} \right) \right] \)

where \( \bar{\mathcal{H}} \) is the target entropy (typically \( -\dim(\mathcal{A}) \))

📖 Notation Guide

Prime (') — next timestep: \(s'\) = next state, \(a'\) = next action

Tilde (~) — freshly sampled from current policy: \(\tilde{a} \sim \pi_\theta(\cdot|s)\)

The stored action \(a\) is from the replay buffer (old policy), while \(\tilde{a}'\) is sampled from the current policy to compute the bootstrap target.

Paper Lineage

Builds On

DDPG
Lillicrap et al., 2015 — Continuous control with deep RL; deterministic actor-critic that SAC extends with stochastic policies
TD3
Fujimoto et al., 2018 — Twin critics + delayed policy updates; SAC adopts twin Q-networks to address overestimation
Soft Q-Learning
Haarnoja et al., 2017 — Maximum entropy framework for RL; direct predecessor using energy-based policies

Built Upon By

CQL
Kumar et al., 2020 — Adds conservative regularizer to SAC for offline RL
REDQ
Chen et al., 2021 — Ensemble of Q-functions with random subset selection; improves SAC's sample efficiency
DrQ / DrQ-v2
Kostrikov et al., 2020/2021 — SAC + data augmentation for vision-based RL; state-of-the-art on pixel-based continuous control
SAC-N / EDAC
An et al., 2021 — SAC with N critics and penalized Q-ensemble for offline RL without explicit conservatism
🔄

Twin Q-Networks

Uses minimum of two Q-values to mitigate overestimation bias (clipped double Q-learning)

📊

Stochastic Policy

Gaussian policy with reparameterization trick: a = tanh(μ + σ·ε), ε ~ N(0,1)

⚖️

Auto Temperature

Automatically tunes α to maintain target entropy, balancing exploration and exploitation
