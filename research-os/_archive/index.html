World Models

Learns a compressed spatial (VAE) and temporal (MDN-RNN) representation of the environment, then trains a tiny controller (867 params) entirely inside dreams using evolutionary optimization (CMA-ES).

VAE + MDN-RNN Dream Training CMA-ES

PlaNet

World Models

Introduces the RSSM (Recurrent State Space Model) and learns to plan entirely in latent space using online CEM optimization, achieving 200x data efficiency over model-free methods.

RSSM CEM Planning Latent Overshooting

Dreamer

World Models

Model-based RL agent that learns long-horizon behaviors from images purely by latent imagination, using a world model and actor-critic trained by backpropagating through learned dynamics.

RSSM Latent Imagination Visual Control
🎲

DreamerV2

World Models

First agent to achieve human-level Atari by learning behaviors inside a world model with discrete categorical latents, KL balancing, and mixed Reinforce + straight-through actor gradients.

Discrete Latents KL Balancing Atari
🌐

DreamerV3

World Models

General algorithm that masters 150+ tasks across 8 diverse domains with fixed hyperparameters using symlog predictions, return normalization, and free bits. First to collect diamonds in Minecraft from scratch.

Symlog Fixed Hyperparams Minecraft
🎬

V-JEPA

World Models

Self-supervised video model that predicts feature representations (not pixels) of masked spatiotemporal regions using a Joint-Embedding Predictive Architecture with ~90% masking.

Feature Prediction JEPA Multi-Block Masking
🤖

V-JEPA 2

World Models

Scales V-JEPA to 1B parameters on 22M videos for state-of-the-art video understanding, then adds an action-conditioned world model (62h robot data) for zero-shot manipulation via energy-based planning.

Scaling SSL Action-Conditioned Robot Planning
